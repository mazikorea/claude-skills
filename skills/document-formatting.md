# 문서 서식 통일 스킬

모든 문서 산출물(엑셀, 워드, PPT, PDF)에 적용하는 서식 기준.

## 1. 컬러 체계 (회색 계열 통일)

| 영역 | 배경색 | 텍스트 |
|------|--------|--------|
| 헤더 행 | #B7B7B7 (진한 회색) | 검정, 볼드 11pt |
| Total/합계 행 | #EFEFEF (연한 회색) | 검정, 볼드 |
| 강조/소계 행 | #D9D9D9 (중간 회색) | 검정 |
| 데이터 셀 | #FFFFFF (흰색) | 검정 10pt |
| 음수 값 | - | 빨간색(#C00000) |

**컬러 금지**: 노란색, 파란색, 주황색 등 유채색 배경 사용 금지. 모든 구분은 회색 명도 차이로만 표현.

## 2. 폰트 및 정렬

- 본문: 맑은 고딕 또는 Arial, 10pt
- 헤더: 볼드, 11pt
- **기본 정렬: 모든 텍스트·숫자·퍼센트 포함 전체 왼쪽 맞춤.**
- 숫자: 왼쪽 정렬, 천 단위 콤마, 소수점 필요시 1자리

## 3. 구성 원칙

### 엑셀/스프레드시트
- **A열(1열): 공란** — 20px 너비, indent용 여백. 실데이터는 B열(2열)부터 시작
- **1행: 공란** — 22px 높이, 여백. 실데이터는 2행부터 시작
- 데이터 시작 위치: **B2** (B열, 2행)
- 상단 헤더: 계층형 병합 (대분류 → 중분류 → 소분류)
- 셀 테두리: 얇은 실선, 섹션 간 굵은 실선 구분
- **눈금선 숨김 필수**: 모든 시트에 `sheet.sheet_view.showGridLines = False` 적용

### 워드/PDF
- 제목: 14pt 볼드
- 소제목: 12pt 볼드
- 본문: 10pt, 줄간격 1.15
- 표: 위 컬러 체계 동일 적용
- 여백: 상하좌우 2.5cm

### PPT
- 제목 슬라이드: 진한 회색(#B7B7B7) 배경 또는 흰색 배경 + 진한 회색 텍스트
- 본문 슬라이드: 흰색 배경
- 표/차트: 위 컬러 체계 동일 적용
- 폰트: Arial 또는 맑은 고딕

## 4. 서식 품질 기준 (깔끔함의 기준선)

### 금지 사항 (과한 서식)
- 셀 배경색 3단계 초과 금지 (진한 회색 / 중간 회색 / 연한 회색만 사용)
- 볼드+이탤릭+밑줄 동시 사용 금지. 강조는 볼드 하나만
- 테두리 이중선, 점선 금지. 실선만 사용
- 그라데이션, 패턴 채우기 금지
- 폰트 2종 초과 금지 (Arial/맑은 고딕 중 하나 + 필요시 고정폭 1종)
- 셀 내 줄바꿈 2회 초과 금지. 내용이 길면 셀을 나눌 것

### 일관성 규칙
- 같은 VIEW(시트/페이지)에서 동일 성격 데이터는 반드시 동일 칼럼 너비 적용
- 같은 시트 내 숫자 포맷 통일: 금액은 모두 원 단위 or 모두 천원 단위 (혼용 금지)
- 퍼센트는 모두 소수점 1자리 통일 (XX.X%)
- 날짜 포맷 통일: 한 문서 내에서 YYYY-MM-DD 또는 YY.MM.DD 중 하나만
- 헤더 행의 텍스트 정렬: 모두 왼쪽 맞춤으로 통일
- 데이터 행: 전체 왼쪽 맞춤 (숫자·퍼센트 포함)

### 칼럼 너비 기준
- 동일 데이터 유형(금액, 비중, 날짜 등)은 같은 너비
- 칼럼 너비는 데이터 최대 길이 + 여백 기준. 과도하게 넓은 빈 칼럼 금지
- 텍스트가 잘리는 것보다 줄바꿈이 나음. 단, 줄바꿈 2회 초과 시 칼럼 분리

### 불필요한 서식 제거
- 빈 행/빈 칼럼으로 간격 조절 금지. 셀 높이/너비로 조절
- 장식용 셀 병합 금지. 병합은 헤더 계층 구조에서만 사용
- 의미 없는 번호 열(1,2,3...) 자동 추가 금지. 사용자가 요청한 경우만
- 워터마크, 배경 이미지 금지

## 5. 다른 스킬과의 관계

sales-report-format, competitor-benchmark 등 다른 스킬에서 문서 산출물을 만들 때, 해당 스킬의 내용 규칙 + 이 스킬의 서식 규칙을 함께 적용한다.
