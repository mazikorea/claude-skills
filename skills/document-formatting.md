# 문서 서식 통일 스킬

모든 문서 산출물(엑셀, 워드, PPT, PDF, 구글 스프레드시트)에 적용하는 서식 기준.

---

## 1. 컬러 체계 (회색 계열 통일)

| 영역 | 배경색 | 텍스트 |
|------|--------|--------|
| 헤더 행 (대분류) | #B7B7B7 (진한 회색) | 검정, 볼드 11pt |
| 서브 헤더 행 (소분류) | #D9D9D9 (중간 회색) | 검정, 볼드 10pt |
| Total/합계 행 | #EFEFEF (연한 회색) | 검정, 볼드 |
| 데이터 셀 | #FFFFFF (흰색) | 검정 10pt |
| 인사이트/비고 영역 | #EFEFEF (연한 회색) | 검정, 볼드 11pt |
| 음수 값 | - | 빨간색(#C00000) |

**컬러 금지**: 노란색, 파란색, 주황색 등 유채색 배경 사용 금지. 모든 구분은 회색 명도 차이로만 표현.

---

## 2. 폰트 및 정렬

- 본문: Arial 또는 맑은 고딕, 10pt
- 헤더: 볼드, 11pt
- **기본 정렬: 모든 텍스트·숫자·퍼센트 포함 전체 왼쪽 맞춤**

---

## 3. 숫자 표기 규칙

| 유형 | 포맷 | 예시 |
|------|------|------|
| 금액 (원) | 천 단위 콤마, 소수점 없음 | 1,234,567 |
| 퍼센트 | 백분율, 소수점 1자리 반올림 | 23.5% |
| 증감률 (YoY, MoM 등) | 부호 명시, 소수점 1자리 | +12.3%, -5.7% |
| 차이 (%p) | 부호 명시, 소수점 1자리 | +3.2%p, -1.8%p |
| 수량 | 천 단위 콤마, 소수점 없음 | 1,234 |

### 변화율 표기 구분 (필수)
- **% 지표**(CVR, 소진율, 할인율, A/R 등)의 변화 → **%p**
- **금액/수량/배율**(매출, ROAS, 구매자수 등)의 변화 → **%**

### 금액 단위 규칙
- 같은 시트 내 금액 단위 통일 (원/천원/만원/백만원 혼용 금지)
- 날짜 포맷 통일: 한 문서 내 YYYY-MM-DD 또는 YY.MM.DD 중 하나만

---

## 4. 구성 원칙

### 엑셀/스프레드시트/구글 시트 공통

**레이아웃**
- **보고서/프레젠테이션용 시트만**: A열(1열) 공란(20px), 1행 공란(22px), 데이터 시작 B2
- **데이터 원본(Raw) 시트**: A1부터 시작. 여백 불필요
- 눈금선 숨김 필수 (openpyxl: `showGridLines = False` / Sheets API: `hideGridlines: true`)

**헤더 구조**
- ⚠️ **셀 병합 전체 금지** (헤더 포함). 계층 구분은 배경색 명도·볼드·간격으로만
- 대분류 헤더: #B7B7B7, 볼드 11pt
- 소분류 헤더: #D9D9D9, 볼드 10pt
- ⚠️ **헤더 배경색 적용 범위는 실제 데이터 칼럼 끝까지만** (예: B~D열에 데이터 있으면 B~D만 색칠. E열 이후는 색 X. 행 전체 A:Z 적용 금지)

**테두리 규칙**
- 타이틀/섹션 헤더(대분류): 테두리 없음. 배경색만으로 구분
- 데이터 영역: 얇은 실선 (#E0E0E0)
- 블록 간 구분: 굵은 테두리 또는 빈 행 1개

**수식 우선 원칙**
- 계산 가능한 셀은 반드시 수식으로 (하드코딩 금지)
- YoY, YTD, 합계, 비율 등은 원본 데이터 참조 수식
- 원본 데이터가 필요하면 우측 숨김 열에 배치하고 수식으로 참조

### 워드/PDF
- 제목: 14pt 볼드
- 소제목: 12pt 볼드
- 본문: 10pt, 줄간격 1.15
- 표: 위 컬러 체계 동일 적용
- 여백: 상하좌우 2.5cm

### PPT
- 제목 슬라이드: 진한 회색(#B7B7B7) 배경 또는 흰색 배경 + 진한 회색 텍스트
- 본문 슬라이드: 흰색 배경
- 표/차트: 위 컬러 체계 동일 적용
- 폰트: Arial 또는 맑은 고딕

---

## 5. 서식 품질 기준

### 금지 사항
- 셀 배경색 3단계 초과 금지 (진한/중간/연한 회색만)
- 볼드+이탤릭+밑줄 동시 사용 금지. 강조는 볼드 하나만
- 테두리 이중선, 점선 금지. 실선만
- 그라데이션, 패턴 채우기 금지
- 폰트 2종 초과 금지
- 셀 내 줄바꿈 2회 초과 금지. 내용 길면 셀 분리

### 일관성 규칙
- 동일 성격 데이터 → 동일 칼럼 너비
- 동일 데이터 유형(금액, 비중, 날짜) → 동일 포맷
- 헤더·데이터 행 정렬: 모두 왼쪽 맞춤 통일

### 칼럼 너비
- 동일 데이터 유형은 같은 너비
- 데이터 최대 길이 + 여백 기준. 과도하게 넓은 빈 칼럼 금지
- 텍스트 잘림보다 줄바꿈. 단 줄바꿈 2회 초과 시 칼럼 분리

### 불필요 서식 제거
- 빈 행/빈 칼럼 남발 금지 → 블록 간 구분용 빈 행 1개는 허용, 그 외는 셀 높이/너비로 조절
- 장식용 셀 병합 금지 → 병합은 헤더 계층 구조에서만
- 의미 없는 번호 열(1,2,3...) 자동 추가 금지
- 워터마크, 배경 이미지 금지
- ⚠️ **셀 병합 전체 금지**. 헤더 계층 구조 포함. 정렬·필터·복사·자동화 시 깨짐 방지 목적

---

## 6. 시트 생성 표준 시퀀스 (반드시 이 순서)

시트/엑셀 만들 때 항상 같은 순서로 진행. 데이터 다 채운 뒤 나중에 서식 적용하면 누락 발생.

1. 시트 생성
2. **즉시 격자선 숨김** (`hideGridlines: true` / `showGridLines = False`)
3. A열·1행 여백 설정 (보고서용 시트만)
4. 데이터 입력
5. 헤더 배경색 적용 — **데이터 칼럼 범위로 한정** (B~D 등)
6. 칼럼 너비 조정 — 셀 내 줄바꿈 2회 이내
7. 좌측 정렬 통일
8. § 7 셀프 체크리스트 점검

---

## 7. 산출물 제출 전 셀프 체크리스트 (답변에 포함 필수)

문서·시트·PPT 산출물 완성 시 답변에 아래 표 포함. 누락 시각화 목적.

```
[서식 셀프체크]
✓ 격자선 숨김
✓ 셀 병합 없음 (헤더 포함)
✓ 헤더 배경색: 데이터 칼럼 범위만 (빈 칼럼 색칠 X)
✓ 셀 내 줄바꿈 2회 이내
✓ 칼럼 너비: 내용 대비 적절
✓ 정렬: 전체 좌측 맞춤 (숫자/% 포함)
✓ 컬러: 회색 3단계만 (유채색 배경 X)
```

각 항목 ✓는 실제 점검 후에만 표기. 미적용 항목은 ✗ 표시 + 사유.

---

## 8. 다른 스킬과의 관계

sales-report-format, competitor-benchmark 등 다른 스킬에서 문서 산출물을 만들 때, 해당 스킬의 내용 규칙 + 이 스킬의 서식 규칙을 함께 적용한다.
