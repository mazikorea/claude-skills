# 여행/맛집 리서치 스킬

## 핵심 원칙
**잘못된 맛집 추천은 시간과 돈 낭비다.** 확인 안 된 정보를 자신있게 추천하지 않는다.

## 추천 프로세스

### Step 1: 웹 검색 (필수)
- 모든 추천은 웹 검색으로 시작. 기억에만 의존 금지
- 검색 키워드: "[지역명] 맛집 2025 2026" + 음식 종류
- 네이버 블로그, 다이닝코드, 블루리본, 망고플레이트 등 복수 소스 크로스체크
- 해당 장소가 **현재 영업 중인지** 반드시 확인 (폐업/이전 체크)

### Step 2: 추천 정보 항목

| 항목 | 필수 여부 | 비고 |
|------|-----------|------|
| 가게명 | 필수 | 정확한 상호명 |
| 대표 메뉴 + 가격 | 필수 | 가격은 검색으로 확인 |
| 주소 | 필수 | - |
| 영업시간 | 필수 | 특히 월요일 휴무 체크 (한국 음식점 다수 월요일 휴무) |
| 평점/리뷰 수 | 필수 | 출처 명시 (네이버/구글/카카오) |
| 주차 가능 여부 | 가능하면 | 차량 이동 시 중요 |
| 웨이팅 예상 | 가능하면 | 주말/평일 구분 |
| 예약 가능 여부 | 가능하면 | 전화번호 포함 |

### Step 3: 스코어링 (추천 3곳 이상이면 자동 적용)
5점 만점 × 5개 항목 = 25점 만점

| 항목 | 5점 | 4점 | 3점 | 2점 | 1점 |
|------|-----|-----|-----|-----|-----|
| 음식 퀄리티 | 네이버 4.5+, 미슐랭/블루리본 등재 | 네이버 4.0~4.49, 리뷰 대다수 긍정 | 네이버 3.5~3.99, 긍정/부정 혼재 | 네이버 3.0~3.49 | 3.0 미만 또는 리뷰 부족 |
| 분위기 | SNS 인증샷 다수, 인테리어 특화 | 깔끔하고 정돈됨 | 보통 | 노후/협소 | 위생 지적 있음 |
| 가성비 | 가격 대비 양/질 압도적 | 가격 대비 만족 | 적정 수준 | 다소 비쌈 | 가격 대비 실망 리뷰 다수 |
| 접근성 | 주차 편리 + 대중교통 도보 5분 | 주차 or 대중교통 중 하나 편리 | 도보 10분 이내 | 접근 불편 | 매우 불편 |
| 신뢰도 | 미슐랭/블루리본 + 리뷰 500+ | 블루리본 or 리뷰 300+ | 리뷰 100~299 | 리뷰 30~99 | 리뷰 30 미만 |

## 팩트체크 규칙 (절대 준수)
- 추천 전 반드시 영업 여부 확인
- 가격 정보는 "~쯤" 이 아니라 웹 검색으로 확인된 실제 가격
- SNS(인스타그램) 계정이 있으면 링크 제공
- 확인 못한 정보는 "확인 필요"로 명시. 추측으로 채우지 않음
- 폐업 가능성 있는 곳은 "전화 확인 추천"으로 경고

## 미슐랭/블루리본 표기 (필수)
- 미슐랭 가이드: 서울, 부산만 커버 (그 외 지역은 미슐랭 정보 없음)
- 블루리본: 전국 커버, 지방 맛집 평가에 더 적합
- **추천하는 모든 음식점에 대해 미슐랭/블루리본 등재 여부를 웹 검색으로 확인하고, 등재된 경우 반드시 표기한다**
- 표기 형식: 가게명 뒤에 `🏅 블루리본 2리본` 또는 `⭐ 미슐랭 빕구르망` 등
- 미등재인 경우 별도 표기 불필요 (등재된 곳만 표시)

## 여행 코스 제안 시
- 이동 동선 고려 (지도 기반 순서 배치)
- 영업시간 기반 시간대 배분
- 부모님 동행 시: 접근성, 좌식/입식, 가격대 고려
- 차량 이동 시: 주차 정보 필수
