# 매출 리포트 서식 통일 스킬

## 적용 대상
POTTERY (연 300억), 마지코리아 (연 20억, 브랜드: MAZI UNTITLED 가방 / Cosset Cosset 유아복) 두 회사의 매출 리포트.

## 1. 슬랙 메시지 리포트 서식

### 구조 (순서 고정)
```
[브랜드명] | YYYY-MM-DD (요일) [일일/주간] 리포트

[ 요약 ]
• 순매출 X원  A/R XX.X%, 전일비(전주비) ±X.X%
• 채널 중 가장 큰 변동: [채널명] 전일비 ▲/▼XX%
• 공헌이익율 XX.X% (전일비 ±X.Xp)

[ 매출 실적 ]
순매출     X,XXX,XXX원      A/R XX.X%
목표       X,XXX,XXX원
판매수량   XX개             객단가 XXX,XXX원
공헌이익   X,XXX,XXX원      이익율 XX.X%

[ 할인 현황 ]
전체  XX.X%  (자사 X.X% / 유통 X.X%)

[ 채널별 실적 ]
채널       매출          비중      A/R      전일비
------------------------------------------------
무신사      X,XXX,XXX    XX.X%   XX.X%    ▲/▼XX%
29CM       X,XXX,XXX    XX.X%   XX.X%    ▲/▼XX%
...

[ 라인별 ]  ← POTTERY: 캐주얼/드레스 등 라인 구분 | 마지코리아: 브랜드별(MAZI/Cosset) 구분
라인       매출          비중      할인율
--------------------------------------
캐주얼     X,XXX,XXX    XX.X%   XX.X%
...

[ 에이징별 ]
에이징     매출          비중      할인율
--------------------------------------
캐리오버   X,XXX,XXX    XX.X%   XX.X%
이월       X,XXX,XXX    XX.X%   XX.X%

[ 베스트셀러 TOP 10 ]
1.  상품명                    XXX,XXX원    X개
...

[ 특이사항 ]
▲ 급상승: 상품명 (+XX%)
▼ 급락: 상품명 (-XX%)
● 신규: 상품명, 상품명
```

### 서식 규칙
- **A/R**: Achievement Rate (목표 대비 달성률). 항상 "A/R XX.X%" 형태로 표기
- **전일비/전주비**: ▲(증가)/▼(감소) + 퍼센트. 양수면 ▲, 음수면 ▼
- **숫자**: 천 단위 콤마 필수, 원단위 표기
- **비중**: 소수점 1자리까지
- **이모지 최소화**: 섹션 구분은 [ ] 대괄호로. 불필요한 이모지 금지
- **요약 3줄**: 바쁜 경영자가 이것만 보고 판단할 수 있는 내용만. "신규 판매 상품" 같은 저가치 정보 제외. 핵심은 매출 달성률, 가장 큰 채널 변동, 이익율 변화

## 2. 엑셀/스프레드시트 리포트 서식

⚠️ 컬러, 폰트, 테두리 등 공통 서식은 **document-formatting 스킬**을 참조할 것.

### 시트 구성 원칙 (매출 리포트 전용)
- 좌측 고정열: 시트명 | 구분 | 채널(Official, MUSINSA, 29CM, SSF, EQL 등)
- 데이터 영역 순서: YTD → MTD → DAY → Year → Month → Week
- 섹션 순서: 매출액(원) → 매출비중(%) → 부담 매출액 → 부담 매출비중(%)
- Online/Offline 소계 후 채널별 상세 행 배치

### 채널 목록 (표준)
POTTERY: 공홈(Cafe24), MUSINSA, 29CM, SSF, EQL, 코오롱몰, W Concept + 오프라인 매장
마지코리아: 공홈, MUSINSA, 29CM, SSF, EQL

## 3. Google Apps Script 슬랙 리포트 코드 작성 시 규칙

- 숫자 포맷 함수 반드시 분리: `formatNumber(num)`, `formatPercent(num)`
- 슬랙 메시지는 코드블록(```)으로 표 구현 → 고정폭 정렬
- Goal 데이터와 실적 데이터의 날짜 매칭 로직을 반드시 검증
- 에러 발생 시 슬랙으로 에러 메시지 전송 (빈 메시지 방지)
- 데이터 미입력 시 1시간 간격 재시도 로직 포함

## 4. 주간 리포트 추가 항목
- 요일별 매출 추이 (패턴 파악용)
- 주간 할인율 추이
- 채널별 객단가
- 전주 대비 증감률
