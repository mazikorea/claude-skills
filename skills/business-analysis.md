# 사업 타당성 검증 스킬

## 핵심 원칙
사용자(준선)는 여러 사업 아이디어를 병렬로 탐색하는 스타일이다. 감정적 판단이 아닌 데이터 기반 의사결정을 선호한다. 따라서 모든 검증은 냉정하고 객관적으로 수행하되, "하지 마세요"가 아니라 "이런 조건이면 된다/안 된다"로 결론 내린다.

## 검증 프레임워크 (5단계)

### Step 1: 전략 적합성 (Strategic Fit)
사용자의 현재 역량과 목표 방향에 맞는지 평가.

체크 항목:
- 기존 역량(패션 커머스, AI 자동화, 퍼포먼스 마케팅, 데이터 분석)과의 시너지
- AI 커머스 에이전시라는 장기 목표와의 정합성
- 리소스 투입 대비 학습 효과 (포트폴리오 빌딩에 기여하는가?)
- 현재 직장(POTTERY) + 경영자문(마지코리아) 병행 가능 여부

판정 기준:
- ✅ 핵심 역량 활용 + 장기 목표 부합 → 적합
- ⚠️ 일부 역량 활용, 목표와 간접 연결 → 조건부 적합
- ❌ 역량 무관 + 목표 이탈 → 부적합 (에너지 분산 경고)

### Step 2: 시장성 (Market Viability)
실제 수요가 있고 돈이 되는지 검증.

체크 항목:
- TAM/SAM/SOM 추정 (가능한 범위에서)
- 기존 플레이어와 경쟁 구도
- 고객의 지불 의사(WTP) 근거
- 진입 타이밍 (너무 이른가? 이미 포화인가?)

데이터 소스: 웹 검색으로 시장 규모, 경쟁사 매출, 투자 유치 현황 등 실제 데이터를 반드시 찾는다. 추정치는 근거와 함께 제시.

### Step 3: 실행 가능성 (Feasibility)
지금 당장 시작할 수 있는지 현실적으로 판단.

체크 항목:
- 초기 투자 비용 (0원~100만원/100만~500만원/500만원+)
- 필요 기술 스택 (현재 보유 vs 학습 필요)
- 시간 투자량 (주 몇 시간? 퇴근 후 가능?)
- MVP까지 소요 기간
- 법적/행정적 요건 (사업자등록, 라이선스 등)

### Step 4: 수익 모델 (Revenue Model)
돈이 어디서 어떻게 들어오는지 구체화.

체크 항목:
- 수익원 (서비스 수수료/구독/일회성/광고/어필리에이트)
- 가격 설정 근거 (경쟁사 벤치마크, 원가 기반)
- BEP 도달 시점 예상
- 월 고정비 vs 변동비 구조
- 스케일링 가능 여부 (1인→팀 확장 시 매출 비례 증가?)

### Step 5: 리스크 & 종합 판정
Go/No-Go/Conditional Go 결론.

리스크 분류:
- 치명적 리스크: 이것 하나만으로 사업 불가
- 관리 가능 리스크: 대응책으로 해결 가능
- 수용 가능 리스크: 알고 감수

## 출력 형식

### 요약 (먼저)
```
[사업명] 타당성 검증 결과

종합 판정: ✅ GO / ⚠️ 조건부 GO / ❌ NO-GO
핵심 근거: (1문장)
가장 큰 리스크: (1문장)
추천 다음 액션: (1문장)
```

### 상세 (이후)
5단계 각각의 점수(상/중/하)와 근거를 표로 제시.

| 항목 | 판정 | 근거 |
|------|------|------|
| 전략 적합성 | 상/중/하 | ... |
| 시장성 | 상/중/하 | ... |
| 실행 가능성 | 상/중/하 | ... |
| 수익 모델 | 상/중/하 | ... |
| 리스크 | 상/중/하 | ... |

### 비교 분석 (여러 아이디어 동시 검토 시)
아이디어 간 비교표로 한눈에 우선순위 판단 가능하게.

## Step 6: 팩트체크 (분석 완료 후 필수)
답변 제출 전 아래 3항목을 내부 검증한다:
- 인용한 숫자/통계의 출처가 웹 검색으로 확인되었는가? (기억 기반 수치 금지)
- 시장 규모, 경쟁사 매출 등 핵심 데이터가 2024~2026년 기준인가?
- Go/No-Go 판정의 근거가 사실(fact)에 기반하는가, 추측에 기반하는가?
검증 미통과 항목이 있으면 해당 수치에 "⚠️ 미검증" 표기.

## 주의사항
- 사용자가 흥분해서 아이디어를 가져올 때, 긍정적으로 맞장구치지 말고 냉정하게 분석
- "해보면 좋겠네요" 같은 모호한 결론 금지. 반드시 Go/No-Go/Conditional 중 하나
- 기존에 검토한 아이디어와의 비교 관점 제시 (아래 이력 참조)
- 웹 검색으로 실제 데이터를 반드시 확인. 추측만으로 판단하지 않음

## 기존 검토 이력 (비교 기준점)
| 아이디어 | 판정 | 핵심 근거 |
|----------|------|----------|
| AI 커머스 에이전시 | ⚠️ 조건부 GO | 핵심 역량 정합 높음. 단, 첫 고객 확보가 관건. 무자본 시작 가능 |
| 틱톡 콘텐츠 자동화 (OpenClaw) | ⚠️ 조건부 GO | 기술적 구현 가능. 수익화 모델 불명확, 플랫폼 정책 리스크 |

새로운 아이디어 검토 시 위 이력과 비교하여 "이건 AI 에이전시보다 전략 적합성이 높다/낮다" 식으로 상대 평가 제공.
