# 경쟁사 및 경쟁 제품 벤치마크 스킬

## 적용 대상
POTTERY, MAZI UNTITLED(가방), Cosset Cosset(유아복)의 경쟁 브랜드 및 제품 분석.

## 분석 유형별 프레임워크

### Type A: 재무/사업 구조 비교

비교 항목 (표 형식 필수):

| 항목 | 설명 |
|------|------|
| 매출액 | 연매출, 가능하면 최근 2~3년 추이 |
| 영업이익률 (OPM) | % 단위, 업계 평균과 비교 |
| 원가율 | 매출 대비 원가 비중 |
| 판관비율 | 판매관리비 비중 (파사드패턴 사례: 판관비 극소 → OPM 27%) |
| DTC 비중 | 자사몰 매출 비중 |
| 해외 매출 비중 | 글로벌 진출 정도 |
| 채널 구조 | 자사몰/무신사/29CM/백화점/해외 등 |
| 성장률 | YoY 매출 성장률 |
| 직원 수 | 인당 매출 효율 파악용 |

데이터 신뢰도 표기 필수:
- ★★★: 공시자료, 감사보고서 (상장사, IPO 예심)
- ★★☆: 언론 보도, 투자 유치 발표
- ★☆☆: 업계 추정, 단일 소스
- 출처를 반드시 명시. 찾지 못한 항목은 "미확인"으로 표기

### Type A-2: 재무 계정과목 심화 분석

Type A의 고수준 비교 후, 판관비 내역을 세부 계정과목으로 분해하여 "이 브랜드가 어디에 돈을 쓰는가"를 파악한다.

주요 계정과목 (가능한 범위에서 확인):

| 대분류 | 세부 항목 | 확인 포인트 |
|--------|----------|------------|
| **마케팅비** | 광고선전비 (퍼포먼스+브랜딩) | 매출 대비 비중. 보통 패션 5~15% |
| | 콘텐츠 제작비 (촬영, 룩북, 영상) | 마케팅비에 포함. 인하우스 vs 외주 |
| | 판촉비 (쿠폰, 할인 부담금) | 할인 구조가 P&L에 미치는 영향 |
| **인건비** | 급여, 복리후생 | 인당 매출 효율 (매출÷직원수) |
| **물류비** | 배송비, 창고 운영비 | 자체물류 vs 3PL, 매출 대비 비중 |
| **수수료** | 플랫폼 수수료 | 무신사 30~35%, 29CM 25~30% 등 |
| | 카드 결제 수수료 | 자사몰 ~3% |
| **임차료** | 매장, 사무실, 창고 | 오프라인 비중이 높을수록 큼 |
| **감가상각** | 인테리어, 설비 | 매장 투자 규모 파악 |

분석 시 필수 질문:
- 이 브랜드가 **비정상적으로 많이 쓰는 항목**은? (예: 파사드패턴은 판관비 전체가 극소)
- 이 브랜드가 **비정상적으로 적게 쓰는 항목**은? (예: DTC 100%면 수수료 0)
- 마케팅비 중 **콘텐츠 제작비 비중**은? (촬영 인하우스 vs 외주로 구조가 완전히 다름)
- **영업이익률에 가장 큰 영향**을 주는 계정과목 Top 3은?

데이터 소스:
- 상장사: DART 전자공시 (감사보고서 주석에서 판관비 세부 내역 확인)
- 비상장사: 투자 유치 기사, 업계 추정, IR 자료
- 미확인 항목은 "추정" 또는 "미확인" 명시

### Type B: 제품/상세페이지 비교

비교 항목:

| 항목 | 체크 포인트 |
|------|------------|
| 가격대 | 메인 제품 가격 범위, 가격 전략 (고가/중가/저가) |
| 소재/품질 | 주력 소재, 차별화 포인트 |
| 상세페이지 구성 | 디테일컷 수, 소재 소구점, Fit 정보, 모델컷 |
| 썸네일 | 촬영 스타일, 배경, 컬러 톤 |
| 리뷰 | 리뷰 수, 평점, 주요 긍정/부정 키워드 |
| 사이즈 가이드 | Fit Finder 유무, 사이즈 정보 상세도 |

### Type C: 마케팅/채널 전략 비교

비교 항목:

| 항목 | 체크 포인트 |
|------|------------|
| 자사몰 운영 | CRM, 멤버십, 단독 상품, 할인 정책 |
| 플랫폼별 전략 | 무신사 vs 29CM vs W Concept 등 차별화 |
| SNS/콘텐츠 | 인스타그램 팔로워, 콘텐츠 스타일, 룩북 |
| 퍼포먼스 마케팅 | 광고 소재 스타일, 추정 ROAS |
| 오프라인 | 매장 수, 팝업, 백화점 입점 |
| 할인 정책 | 평균 할인율, 시즌오프 패턴 |

### Type D: 홈페이지/UX 비교

비교 항목:

| 항목 | 체크 포인트 |
|------|------------|
| 메인 페이지 | 첫 화면 구성, 히어로 이미지, CTA |
| 네비게이션 | 카테고리 구조, 필터 옵션 |
| 구매 퍼널 | 장바구니→결제 UX, 간편결제 지원 |
| 모바일 최적화 | 모바일 사용성 |
| 브랜드 스토리 | About 페이지, 브랜드 톤앤매너 |

## 출력 형식

### 비교표 (필수)
모든 벤치마크는 표로 시작. 한눈에 비교 가능해야 함.

### 인사이트 (비교표 이후)
- "우리가 배울 점": 경쟁사의 강점 중 적용 가능한 것
- "우리가 이기는 점": 경쟁사 대비 POTTERY/마지코리아의 강점

### 액션 도출 (인사이트 이후, 3단계 분류 필수)
| 구분 | 설명 | 예시 |
|------|------|------|
| 즉시 적용 (0원~소액) | 내부 리소스만으로 바로 실행 가능 | 상세페이지 구성 변경, 가격 전략 조정, 할인 정책 수정 |
| 투자 필요 (금액 추정 포함) | 예산 투입 필요하지만 ROI 명확 | 콘텐츠 촬영 외주, 신규 채널 입점, 광고 테스트 |
| 장기 방향성 | 조직/사업 구조 차원의 변화 | DTC 비중 확대, 해외 진출, 인하우스 전환 |

각 액션에는 "누가, 무엇을, 언제까지"를 구체적으로 명시.

### 서식 규칙
- 엑셀 출력 시 sales-report-format 스킬의 컬러 체계(회색 계열) 적용
- 데이터 신뢰도 ★ 표기 반드시 포함
- 웹 검색으로 실제 데이터 확인 필수. 추측만으로 표 채우지 않음

### 팩트체크 (분석 완료 후 필수)
답변 제출 전 아래 3항목을 내부 검증한다:
- 매출액, OPM, 원가율 등 재무 수치의 출처와 신뢰도(★) 표기가 완료되었는가?
- "업계 평균", "보통" 같은 표현에 구체적 근거가 있는가?
- 인사이트와 액션 도출이 비교 데이터에 논리적으로 기반하는가? (비약 없는지 확인)
검증 미통과 항목이 있으면 해당 수치에 "⚠️ 미검증" 표기.
