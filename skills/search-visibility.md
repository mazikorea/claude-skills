# 브랜드 검색 가시성 통합 최적화 스킬

---

## 핵심 개념: SEO / GEO / 플랫폼 검색 차이

| 구분 | 대상 | 설득 대상 | 핵심 신호 | 내 통제력 |
|------|------|---------|---------|:--------:|
| **구글 SEO** | 구글 검색결과 순위 | 구글 랭킹 알고리즘 | 백링크(r=0.9), 콘텐츠 품질, E-E-A-T, CWV | 높음 |
| **네이버 SEO** | 네이버 통합검색·쇼핑탭 | 네이버 GBDT 알고리즘 | 적합도(키워드), 인기도(최신성), 신뢰도 | 중간 |
| **GEO** | ChatGPT·Perplexity·AI Overview | AI 인용·추천 판단 | 브랜드 검색량, 제3자 PR, 리스티클 형식 | 낮음~중간 |
| **무신사/29CM** | 플랫폼 내부 검색·추천 | 플랫폼 자체 알고리즘 | 매출×판매량, 후기수, 상품명 적합도 | 낮음 |

**3가지 핵심 사실**
1. 무신사/29CM 랭킹은 SEO와 완전히 독립. 광고도 영향 없음. 판매 데이터 축적만이 답
2. 네이버는 구글과 반대 — 구글은 외부 백링크 중심, 네이버는 자사 플랫폼(스마트스토어·블로그) 우선
3. 공통 레버 4개 — 스키마 마크업, E-E-A-T, PR/미디어, 리뷰·UGC가 복수 채널 동시 효과

## 통합 평가 프레임워크 — 9개 채점 항목

"검색 가시성 점검해줘" 요청 시 아래 9개 항목을 10점 만점으로 실시간 채점.
**반드시 WebFetch·WebSearch로 현재 상태 직접 확인. 기억·추정 기반 채점 금지.**
확인 불가 항목은 점수 없이 "미확인"으로 표기.

---

### 1. 기술 기반
확인 항목:
- robots.txt: GPTBot·ClaudeBot·PerplexityBot 허용 여부 (직접 fetch)
- sitemap.xml: 존재 + Google Search Console·네이버 서치어드바이저 제출 여부
- Title 태그: 홈 + 상품 페이지 각 1개 직접 fetch해서 존재·품질 확인
- Meta Description: 존재 여부 확인
- Canonical 태그: 상품 옵션 URL 중복 관리 여부
- Core Web Vitals: PageSpeed Insights 점수 (pagespeed.web.dev에서 확인)

근거: Title/Meta 누락 시 구글·네이버 CTR 직격. CWV는 구글 tie-breaker. Canonical 없으면 Cafe24 상품 옵션마다 생기는 중복 URL로 크롤 낭비

---

### 2. 구조화 데이터 (Schema)
확인 항목:
- Google Rich Results Test (search.google.com/test/rich-results) 결과
- JSON-LD 스키마 종류: Product·Organization·FAQ·Breadcrumb·AggregateRating 존재 여부
- 상품 가격·재고·평점이 구조화 데이터에 포함됐는지

근거: Schema 구현 시 구글 CTR +30%, AI 인용 73%↑. Product Schema 없으면 구글 쇼핑탭 노출 불가 (복수 독립 연구 일치 ✅)

---

### 3. 카테고리 키워드 전략 (비브랜드 검색)
확인 항목:
- 브랜드를 모르는 신규 고객의 카테고리 검색에 등장 여부
  - POTTERY: "남성 기본 셔츠 추천", "미니멀 남성복 브랜드"
  - MAZI: "국내 미니멀 가방 브랜드", "여성 가방 추천"
  - COSSET: "프리미엄 유아복 추천", "신생아 옷 브랜드"
- 구글·네이버 각각 검색해서 1페이지 내 등장 여부 확인

근거: SEO의 본질은 브랜드 모르는 신규 고객 유입. 브랜드 직검색 노출은 당연한 것. 카테고리 키워드가 진짜 경쟁

---

### 4. 콘텐츠 품질·E-E-A-T·신선도
확인 항목:
- 자사 블로그/매거진 존재 여부 + 최근 업데이트 날짜
- 저자 정보 표기 여부 (사진+약력)
- 상품 설명 깊이 (소재·용도·사이즈·관리법 포함 여부)
- 콘텐츠 최근 갱신 주기 (월 1회 이상 권장)

근거: 구글 E-E-A-T 강화 → 상위3위 확률 +30% (SEMrush ✅). Perplexity 신선도 가중치 40% (비공식 ⚠️). AI 인용의 85%가 최근 2년 내 발행 (복수 연구 ✅)

---

### 5. 외부 권위 (PR + 백링크)
확인 항목:
- 패션 전문지(어패럴뉴스, 패션비즈) 기사 수 (웹 검색)
- 무신사·29CM 뉴스룸 언급 수
- 외부 도메인에서 자사몰 링크 존재 여부
- 링크 없는 브랜드명 언급(Unlinked Mentions) 수

근거: 구글 백링크 r=0.9 (Ahrefs 10억+ 페이지 ✅). GEO에서 자사 4.5% vs PR 59.8% (Amico 사례 1건 ⚠️ 단일 사례)

---

### 6. 내부 링크 & 사이트 아키텍처
확인 항목:
- 홈→카테고리→상품 계층 구조가 명확한지
- 관련 상품 위젯 존재 여부
- 홈페이지 주요 링크가 상품/카테고리로 연결되는지 (vs 로그인/약관 등)
- 브레드크럼 네비게이션 존재 여부

근거: 상위 페이지는 하위 페이지보다 내부 링크 73% 더 많음 (Searchmetrics). 필라-클러스터 구현 시 AI 인용율 +342%, 키워드 랭킹 +63% (90일, ⚠️ 단일 연구)

---

### 7. 네이버 전용 최적화
확인 항목:
- 네이버 서치어드바이저 등록 여부 (searchadvisor.naver.com)
- 네이버 스마트스토어 운영 여부 (Cafe24보다 네이버 자체 플랫폼 우선 노출)
- 네이버 공식 블로그 존재 + C-Rank 구축 여부 (주 2~3회 포스팅)
- 상품명 50자 이내, 핵심 키워드 포함, 중복/어뷰징 없음
- 네이버 검색에서 카테고리 쿼리 노출 여부

근거: 네이버 한국 검색 점유율 60~70%. 2025년 알고리즘 개편 후 경험 기반 콘텐츠·신선도 우대. Cafe24 자사몰은 스마트스토어 대비 구조적 노출 불리 (네이버 서치어드바이저 공식 ✅)

---

### 8. 플랫폼 최적화 (무신사/29CM)
확인 항목:
- 무신사: 입점 여부 + 상품 수 + 리뷰 수 + 판매 순위 추정
- 29CM: 입점 여부 + 브랜드 페이지 완성도
- 상품명 키워드 최적화 (형태소 분석 기반)
- 재고 상시 유지 여부 (품절 시 랭킹 자동 하락)

근거: 무신사 랭킹 = 매출(가격×수량)+후기+CTR. 광고와 완전 독립 (MUSINSA Tech 공식 ✅). 리뷰 5개 상품은 0개 대비 구매확률 270% 높음 (Fera.ai ✅)

---

### 9. 소셜·UGC + 비교·리스티클 콘텐츠
확인 항목:
- 인스타그램 팔로워 수 + 게시물 수
- 브랜드 해시태그 게시물 수
- 커뮤니티 자연 언급 (에펨코리아, 어미새, 블라인드, 맘카페 등)
- "브랜드명 vs", "브랜드명 추천" 외부 비교글 존재 여부
- 자사 블로그에 "Top N" 형식 가이드 존재 여부

근거: Perplexity 인용의 46.7%가 Reddit 등 커뮤니티 (복수 연구 ✅). 전체 AI 인용의 74.2%가 리스티클 형식 (GenOptima ⚠️ 벤더연구). UGC = AI 인용의 21.74% (복수 연구 ✅)
